# covid-19-th

* ติดตามสถานการณ์โรคติดเชื้อโคโรนา 2019 (COVID-19) ผ่านหน้าเว็บ
* ทั้งนี้แหล่งที่มาของข้อมูลใช้จาก API ของกรมควบคุมโรค แบบ Real-Time
* รายงาน covid-19 ประจำวัน https://data.go.th/dataset/covid-19-daily
